# Support

Open Wrangler is experimental prerelease software. For usage questions, reproducible defects, or feature requests, open a [GitHub issue](https://github.com/Matt17BR/openwrangler/issues/new/choose).

Before reporting a defect, include:

- Open Wrangler, editor, operating-system, Python, and dataframe-engine versions
- the input format and a minimal synthetic reproduction when possible
- the relevant **Open Wrangler** output-channel diagnostics
- whether the workspace is local, remote, or untrusted

Never attach proprietary data, credentials, notebook secrets, or full environment dumps. Use GitHub's private vulnerability-reporting channel for security issues as documented in [SECURITY.md](SECURITY.md).
